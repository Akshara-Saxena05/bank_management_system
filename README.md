# Bank Management System

A Bank Management System developed using Java and MySQL that simulates basic banking operations. This project allows users to perform essential banking functionalities through a simple interface.

## Features
- Deposit Money
- Withdraw Money
- Fast Cash
- Balance Enquiry
- Mini Statement
- PIN Change

## Technologies Used
- Java
- MySQL

## Project Objective
The main objective of this project is to simulate real-world banking operations and provide users with a simple banking management system.

## Functionalities
- Users can deposit money into their account
- Withdraw money securely
- Access fast cash options
- Check account balance
- View mini statement (transaction history)
- Change PIN

## Database
MySQL is used to store user and transaction details.

## How to Run the Project
1. Clone the repository
2. Open the project in your Java IDE (IntelliJ IDEA / Eclipse / NetBeans)
3. Configure MySQL database
4. Run the project
